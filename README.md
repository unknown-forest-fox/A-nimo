This is one of the univeristy projects that me and my colleagues work on.

A-nimo is a desk companion designed to help users maintain their focus
while working. The companion is more focused to be used by young people,
as it can be seen in the modern days, they are the easiest distracted while
working/studying.


The design is user friendly, made so the user would want to keep it on
the desk, by its side and use it. The display with eye expressions provides a
feeling of comfort and probably can soothe loneliness.
Once connected to the laptop, A-nimo will continuously monitor the
user, making sure that their focus is on the work on laptop. Once the user is
distracted for too long, the prototype will react by switching eye expressions,
showing itself as being angry at the user for not working, providing a physical,
ambient feedback signal.

The system combines the three components required: JavaScript, physical
components and the Machine Learning part.
